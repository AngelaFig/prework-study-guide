HTML

  • HTML is one of the three cornerstone languages used in web development.
  • HTML elements work together to create the structure of a webpage.
  • Two elements are especially important in understanding how the structure works:the head and body elements.
  • The head element contains information about the webpage, or metadata, which helps the browser render the webpage correctly.
  • The body element represents the visible content shown to the user.

CSS

  • Cascading Style Sheets or CSS is one of the three cornestone languages used in web development.
  • CSS is the language used to describe the presentation of a website that is created with HTML.
  • CSS is also used to descibe how each element on a webpage is rendered.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.

Git

  • Git is a free, open source version control system that lets you manage and keep track of each version of your project.
  • Git allows web developers to collaborate on projects without overwriting eachother's work.
  • Git allows developers to revert to previous versions of the code if a problem is discovered.
  • To access Git, we use the command line and start our command with "git".
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Using git commands, we can initiate a repo, add code to our repo, and view the different versions of our code.
  • Git is great for tracking changes locally on your computer, but to collaborate with others you must use a Git-based repository hosting service, such as GitHub.

JavaScript

  • JavaScript is one of the three cornerstone languages used in web development.
  • JavaScript is a lightweight programming language that is used to provide logic and interactivity for websites.
  • JavaScript adds dynamism to the webpage, thus enhancing the user's experience.
